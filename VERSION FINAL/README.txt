===Instalación, desinstalación y ejecución===

Tanto para la instalación como la desinstalación de la aplicación se necesitan tener permisos de superusuario. De no tenerlos, el sistema mostrará un mensaje de error y no se ejecutará.

Una vez instalada la aplicación y siendo superusuario, se puede llamar al programa con la orden "customgrub2".
